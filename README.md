# rangi
This is a package to colorize your terminal text messages.

## Install & Use
You can install it by:
`npm i rangi`

Then use it like this:
```
import 'rangi'

const yourString = 'your string'
const coloredString = yourString.color('red')
console.log(coloredString)
```
